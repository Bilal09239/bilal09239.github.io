# Linux and GitHub Learning Page

This repository contains a small learning project about **Linux installation** and **GitHub basics**.

The website is hosted using **GitHub Pages** and written in **Markdown**.

## Website

You can view the website here:

➡ https://bilal09239.github.io/

## Project Overview

The website consists of three pages:

### Home Page
Introduces the project and explains the purpose of the website.

### Linux Installation
A beginner-friendly guide explaining how to install Linux on a computer, including:
- what Linux is
- choosing a distribution
- downloading an ISO file
- creating a bootable USB drive
- installing the operating system

### GitHub Basics
An introduction to GitHub and Git, including:
- repositories
- version control
- basic Git commands
- uploading and managing code
- using GitHub Pages

## Technologies Used
