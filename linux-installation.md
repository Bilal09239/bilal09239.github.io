# Linux Installation Guide

This page explains how to install Linux on your computer.  
Linux is a free and open-source operating system and can be used as an alternative to Windows or macOS.

Many people use Linux because it is:

- free to use
- secure and stable
- customizable
- widely used for development and servers

---

# What is a Linux Distribution?

Linux itself is only the **kernel**, which is the core of the operating system.  
Most users install a **Linux distribution (distro)**.

A distribution contains:

- the Linux kernel
- a desktop environment
- system tools
- applications

Popular distributions include:

- Ubuntu
- Linux Mint
- Fedora
- Debian

For beginners, **Ubuntu** or **Linux Mint** are often recommended.

---

# Step 1: Download Linux

First you need to download the operating system.

1. Visit the website of a Linux distribution
2. Download the **ISO file**
3. Save it on your computer

An ISO file is an image of the installation system.

---

# Step 2: Create a Bootable USB Stick

To install Linux you need a **bootable USB drive**.

Common tools:

- Rufus
- Balena Etcher
- Ventoy

Steps:

1. Insert a USB stick (8GB or larger)
2. Open the program
3. Select the ISO file
4. Select your USB device
5. Start the process

This will create a bootable installer.

---

# Step 3: Boot from the USB Stick

Now restart your computer and boot from the USB drive.

During startup press a **boot menu key**, for example:

- F12
- F10
- ESC
- DEL

Select the USB device from the boot menu.

---

# Step 4: Start the Installer

After Linux loads you usually see two options:

- Try Linux
- Install Linux

Choose **Install Linux**.

The installer will guide you through:

- language selection
- keyboard layout
- network setup
- disk configuration
- user account creation

---

# Step 5: Finish Installation

After installation:

1. Restart the computer
2. Remove the USB stick
3. Start your new Linux system

---

# Conclusion

Linux is a powerful operating system that is widely used in development, servers, and personal computers.  
Installing Linux is relatively easy and usually takes less than an hour.
